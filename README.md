=============
promoproducts
=============

----------
Setting Up
----------

Clone this repository:

```
$ git clone https://github.com/leandromuto/promoproducts.git
$ cd promoproducts
```

Assuming you have `virtualenv` and `pip` installed:

```
$ virtualenv env
$ source env/bin/activate
$ pip install -r requirements.txt
```

To run tests:

```
$ nosetests
```
